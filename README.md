# StoreFleet – Backend E-commerce API

## 📌 Overview

StoreFleet is a backend-focused e-commerce project built as part of a learning journey to understand real-world backend development concepts. The project demonstrates how an online store backend works, including authentication, product handling, orders, and reviews.

This repository is intended for educational and practice purposes.

---

## 🚀 Features Implemented

- User authentication & authorization (JWT)
- Role-based access control (Admin / User)
- Product management (CRUD)
- Product search, filtering & pagination
- Review system with ownership checks
- Order placement workflow
- Stock availability checks & updates
- Order tracking (user & admin)
- Secure API structure using controllers & repositories

---

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

---

## 🧠 What I Learned

- Structuring backend projects cleanly
- Building RESTful APIs
- Handling complex query filters
- Implementing authorization logic
- Managing relations between users, products, orders, and reviews
- Writing maintainable controller & repository layers
